
"Random" Photos Screensaver
---------------------------

In order to deploy this screen saver, you must rename RELEASE build of the screen saver executable to have a ".scr" extension:
	1.  Go to bin\Release folder of your project
	2.  Rename the .exe to .scr  

To install on a client machine:
	1.  Copy the .scr file to convenient location on your C: drive.
	2.  Right click the .scr file.
	3.  Select Install.

To uninstall delete the .scr file.

Command-line Arguments
	<no args>  Display screen saver
	"/s"	   Display screen saver
	"/c"	   Show settings window
	"/p"       Preview
